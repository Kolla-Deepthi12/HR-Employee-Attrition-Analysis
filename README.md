# HR Employee Attrition – End-to-End Data Analysis

## Project Overview
This project focuses on analyzing employee attrition using an end-to-end data analytics approach.
The analysis covers data cleaning, SQL-based ETL, and interactive dashboards.

## Tools & Technologies
- Excel – Data cleaning and preparation
- SQL (MySQL) – ETL, transformations, and insights
- Tableau – Data visualization
- GitHub – Version control

## Workflow
1. Raw HR data cleaned using Excel
2. Cleaned data loaded into MySQL
3. SQL used for ETL and insight generation
4. Dashboards built in Tableau

## Key Insights
- Higher attrition observed in Sales department
- Younger employees show higher attrition
- Income and experience influence employee turnover

## Files Included
- Raw and cleaned datasets
- SQL insights and ETL scripts
- Tableau dashboard
